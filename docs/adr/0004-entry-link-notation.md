# ADR-0004: エントリ間リンク記法
Status: accepted
Date: 2026-04-03

## Context
エントリ間リンクの記法として UUID / タイトル / 日付 / 複合フォールバック の4案があった。

## Decision
**`[[タイトル]]` (Obsidian互換) を採用。**

退けた選択肢:
- `[[UUID_PREFIX]]`: 厳密だがユーザーが記憶・入力できない
- `[[YYYY-MM-DD]]`: 同日複数エントリで曖昧
- 複合フォールバック: 解決ロジックが複雑化

タイトル重複時はリスト表示で選択。$EDITOR内で `[[` 入力後にvimカスタム補完関数でタイトル候補を表示。補完用一覧は /dev/shm 上の一時ファイルに書き出し、編集後zeroize削除。

## Consequences
- Obsidianからの移行が容易（リンク記法がそのまま通る）
- タイトル変更時にリンク切れが発生する → 将来的にリネーム時の自動更新を検討
- vim/neovim以外のエディタでは補完が効かない（フォールバックは手動入力）

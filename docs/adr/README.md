# Architecture Decision Records

セキュリティ上重要な設計判断を記録する。フォーマット:

```
# ADR-NNNN: タイトル
Status: accepted | superseded by ADR-XXXX | deprecated
Date: YYYY-MM-DD

## Context
判断が必要になった背景

## Decision
何を選び、何を退けたか

## Consequences
この判断によるトレードオフ
```

---

### Q2: TOMLパースのクレート選定

**カテゴリ**: 技術選定
**背景**: vault.toml / config.toml のパースに使用するクレートを確定する必要がある。serdeベースの型安全アプローチか、手動パースか。

**回答**: serde + toml（型安全）

**信頼性への影響**: REQ-004 (vault.toml) / REQ-005 (config.toml) のパース方式が確定。serde derive による型安全なデシリアライズ/シリアライズが採用された。

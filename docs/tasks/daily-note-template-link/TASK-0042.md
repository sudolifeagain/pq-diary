## タスク概要

`core/src/template.rs` に `TemplatePlaintext` 型と `TemplateMeta` 型を定義し、テンプレートのCRUD関数（create_template, list_templates, get_template, delete_template）を実装する。テンプレートはvault.pqd内に `RECORD_TYPE_TEMPLATE (0x02)` として暗号化格納される。`TemplateName` バリデーション型も定義する。`core/src/error.rs` に `TemplateNotFound` と `InvalidTemplateName` エラーバリアントを追加する。

## 実装詳細

### 1. DiaryError拡張 (core/src/error.rs) 🔵

**信頼性**: 🔵 *EDGE-002, EDGE-003, EDGE-101より*

```rust
#[error("template not found: {0}")]
TemplateNotFound(String),

#[error("invalid template name: {0}")]
InvalidTemplateName(String),
```

### 2. TemplatePlaintext + TemplateMeta (core/src/template.rs) 🔵

**信頼性**: 🔵 *設計ヒアリング: 専用型採用決定*

```rust
#[derive(Debug, Serialize, Deserialize, PartialEq, Zeroize, ZeroizeOnDrop)]
pub struct TemplatePlaintext {
    pub name: String,
    pub body: String,
}

pub struct TemplateMeta {
    pub uuid_hex: String,
    pub name: String,
    pub created_at: u64,
    pub updated_at: u64,
}
```

### 3. TemplateName バリデーション型 🟡

**信頼性**: 🟡 *EDGE-101から妥当な推測*

- 1〜128文字
- スペース禁止
- 空文字禁止
- `new(s: &str) -> Result<TemplateName, DiaryError>`

### 4. CRUD関数 🔵

**信頼性**: 🔵 *entry.rs の既存CRUDパターンに準拠*

- `create_template`: TemplatePlaintext → serde_json → encrypt → write_vault (RECORD_TYPE_TEMPLATE)
- `list_templates`: read_vault → filter(type==0x02) → decrypt → extract name → TemplateMeta
- `get_template`: read_vault → filter(type==0x02) → decrypt → find by name → TemplatePlaintext
- `delete_template`: read_vault → remove matching template → write_vault

## 単体テスト要件

### TC-042-01: テンプレート作成・復号ラウンドトリップ 🔵

### TC-042-02: テンプレート名バリデーション 🔵
**Given**: 空文字, 129文字, スペース含む文字列
**When**: TemplateName::new()
**Then**: DiaryError::InvalidTemplateName

### TC-042-03: 存在しないテンプレートのget 🔵
**Given**: テンプレートなしVault
**When**: get_template("nonexistent")
**Then**: DiaryError::TemplateNotFound

### TC-042-04: テンプレート削除 🔵
**Given**: テンプレートが1つあるVault
**When**: delete_template("daily") → list_templates
**Then**: 空リスト

### TC-042-05: TemplatePlaintext のzeroize検証 🔵
**Given**: TemplatePlaintext を ManuallyDrop で保持
**When**: zeroize() 呼び出し
**Then**: フィールドがゼロクリア

### TC-042-06: エントリとテンプレートの共存 🔵
**Given**: エントリ3件 + テンプレート2件のVault
**When**: list_templates() + list_entries()
**Then**: テンプレート2件、エントリ3件がそれぞれ正しく返る

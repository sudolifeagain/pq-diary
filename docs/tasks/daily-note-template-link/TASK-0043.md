## タスク概要

`core/src/template_engine.rs` にテンプレート変数の抽出・展開ロジックを実装する。`{{var_name}}` パターンをパースし、基本変数（date, datetime, title）の自動設定と、カスタム変数の抽出を行う。

## 実装詳細

### 1. 変数パーサー 🔵

**信頼性**: 🔵 *REQ-112〜113より*

正規表現 `\{\{(\w+)\}\}` で `{{var_name}}` を抽出。
- `date`, `datetime`, `title` → `VariableKind::Builtin`
- それ以外 → `VariableKind::Custom`

### 2. 展開関数 🔵

**信頼性**: 🔵 *REQ-112〜114より*

`HashMap<String, String>` で変数名→値のマッピングを受け取り、`{{var_name}}` を値に置換。未定義の変数はそのまま残す。

## 単体テスト要件

### TC-043-01: 基本変数 {{date}} の抽出 🔵
**Given**: "日付: {{date}}"
**When**: extract_variables()
**Then**: VariableRef { name: "date", kind: Builtin }

### TC-043-02: カスタム変数の抽出 🔵
**Given**: "PJ: {{project_name}}"
**When**: extract_variables()
**Then**: VariableRef { name: "project_name", kind: Custom }

### TC-043-03: 基本変数の展開 🔵
**Given**: "日付: {{date}}" + vars: {"date": "2026-04-05"}
**When**: expand()
**Then**: "日付: 2026-04-05"

### TC-043-04: 同一変数の複数出現 🔵
**Given**: "{{date}} メモ\n---\n{{date}} 終了"
**When**: expand() with date=2026-04-05
**Then**: "2026-04-05 メモ\n---\n2026-04-05 終了"

### TC-043-05: 変数なしテンプレート 🔵
**Given**: "固定テキスト"
**When**: extract_variables()
**Then**: 空Vec

### TC-043-06: 空中括弧 {{}} の処理 🟡
**Given**: "テスト {{}}"
**When**: extract_variables()
**Then**: 空Vec（\w+ にマッチしないため）

### TC-043-07: 複数種類の変数混在 🔵
**Given**: "{{date}} - {{project_name}} by {{title}}"
**When**: extract_variables()
**Then**: 3つの VariableRef (Builtin, Custom, Builtin)

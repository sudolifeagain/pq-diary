## タスク概要

`core/src/link.rs` に `[[タイトル]]` リンクのパーサーを実装する。正規表現 `\[\[([^\[\]]+)\]\]` で本文中のリンクを抽出し、`ParsedLink` として返す。

## 実装詳細

### 1. ParsedLink型 🔵

**信頼性**: 🔵 *ADR-0004・types.rsより*

```rust
pub struct ParsedLink {
    pub title: String,
    pub start: usize,
    pub end: usize,
}
```

### 2. parse_links関数 🔵

**信頼性**: 🔵 *ADR-0004より*

正規表現: `\[\[([^\[\]]+)\]\]`
- `[^\[\]]+` で中身にブラケットを含まないマッチ
- 空マッチ（`[[]]`）は `[^\[\]]+` (1文字以上) なので自動除外

## 実装詳細

### 1. cmd_template_add 🔵

**信頼性**: 🔵 *REQ-101・cmd_new パターンより*

1. resolve_vault_path() → get_password()
2. DiaryCore::new() → unlock()
3. 既存テンプレート確認（同名なら上書き確認）
4. write_template_file(tmpdir) で空ファイル作成
5. launch_editor() でエディタ起動
6. read_template_file() で内容読み取り
7. new_template(name, body) で保存
8. lock()

### 2. cmd_template_list 🔵

**信頼性**: 🔵 *REQ-102・cmd_list パターンより*

1. resolve_vault_path() → get_password()
2. DiaryCore::new() → unlock()
3. list_templates() → アルファベット順ソート → 表示
4. lock()

### 3. cmd_template_show 🔵

**信頼性**: 🔵 *REQ-103・cmd_show パターンより*

1. resolve_vault_path() → get_password()
2. DiaryCore::new() → unlock()
3. get_template(name) → 内容表示
4. lock()

### 4. cmd_template_delete 🟡

**信頼性**: 🟡 *REQ-104・cmd_delete パターンから妥当な推測*

1. resolve_vault_path() → get_password()
2. DiaryCore::new() → unlock()
3. get_template(name) でテンプレート名を表示
4. force でなければ確認プロンプト
5. delete_template(name) で削除
6. lock()

### 5. ディスパッチ接続 🔵

**信頼性**: 🔵 *既存main.rsパターンより*

```rust
Commands::Template { subcommand } => match subcommand {
    TemplateCommands::Add { name } => commands::cmd_template_add(cli, name.clone()),
    TemplateCommands::List => commands::cmd_template_list(cli),
    TemplateCommands::Show { name } => commands::cmd_template_show(cli, name.clone()),
    TemplateCommands::Delete { name } => commands::cmd_template_delete(cli, name.clone(), false),
},
```

## 単体テスト要件

### TC-047-01: template list の表示フォーマット 🔵
**Given**: テンプレート "daily", "meeting", "weekly" が登録済み
**When**: cmd_template_list()
**Then**: アルファベット順で "daily", "meeting", "weekly" と表示

### TC-047-02: template show の出力 🔵
**Given**: テンプレート "daily" (body: "## 振り返り") が登録済み
**When**: cmd_template_show(cli, "daily")
**Then**: "## 振り返り" が表示される

### TC-047-03: 存在しないテンプレートのshow 🔵
**Given**: テンプレートなしVault
**When**: cmd_template_show(cli, "nonexistent")
**Then**: DiaryError::TemplateNotFound

## 実装詳細

### 1. clap フラグ追加 🔵

**信頼性**: 🔵 *REQ-111より*

```rust
New {
    title: Option<String>,
    #[arg(short, long)]
    body: Option<String>,
    #[arg(short, long = "tag")]
    tag: Vec<String>,
    #[arg(long)]
    template: Option<String>,  // NEW
}
```

### 2. cmd_new テンプレート展開 🔵

**信頼性**: 🔵 *REQ-111〜114・dataflow.mdより*

1. `--template` 指定時: get_template(name) でテンプレート取得
2. extract_variables(body) でカスタム変数を抽出
3. 基本変数を自動設定: date=chrono::Local::now().format("%Y-%m-%d"), datetime, title
4. カスタム変数: print!("{var_name}: ") → stdin.read_line() でプロンプト入力
5. expand(body, vars) で展開
6. 展開結果を初期本文としてエディタ起動

### 3. ディスパッチ更新 🔵

**信頼性**: 🔵 *既存パターンより*

```rust
Commands::New { title, body, tag, template } => {
    commands::cmd_new(cli, title.clone(), body.clone(), tag.clone(), template.clone())
}
```

## 注意事項

- chrono クレートは既存依存。format は "%Y-%m-%d" (date), "%Y-%m-%d %H:%M:%S" (datetime)
- カスタム変数プロンプトはCLI固有ロジック（core/に入れない）
- --template と --body の同時指定時: --body 優先（テンプレートを無視）

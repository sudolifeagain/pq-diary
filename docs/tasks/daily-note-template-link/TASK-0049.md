## タスク概要

`cli/src/commands.rs` に `cmd_today` を実装する。当日の `YYYY-MM-DD` タイトルのエントリを検索し、存在すれば edit と同様に開き、存在しなければ `daily` テンプレート（存在する場合）を適用して新規作成する。

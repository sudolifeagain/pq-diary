## 実装詳細

### 1. ルート Cargo.toml 🔵

**信頼性**: 🔵 *PRDセクション2.1・architecture.mdより*

```toml
[workspace]
members = ["core", "cli"]
resolver = "2"

[workspace.package]
edition = "2021"
rust-version = "1.94"
license = "GPL-3.0-or-later"
```

**実装ファイル**: `Cargo.toml`

### 2. core/Cargo.toml 🔵

**信頼性**: 🔵 *architecture.md依存関係セクションより*

```toml
[package]
name = "pq-diary-core"
version = "0.1.0"
edition.workspace = true
rust-version.workspace = true
license.workspace = true

[dependencies]
thiserror = "2"
zeroize = { version = "1", features = ["derive"] }
secrecy = "0.10"
```

**実装ファイル**: `core/Cargo.toml`

### 3. cli/Cargo.toml 🔵

**信頼性**: 🔵 *architecture.md依存関係セクションより*

```toml
[package]
name = "pq-diary"
version = "0.1.0"
edition.workspace = true
rust-version.workspace = true
license.workspace = true

[dependencies]
pq-diary-core = { path = "../core" }
clap = { version = "4", features = ["derive"] }
anyhow = "1"
```

**実装ファイル**: `cli/Cargo.toml`

### 4. core/src/ モジュールスケルトン 🔵

**信頼性**: 🔵 *architecture.mdモジュール構成表・ヒアリングQ1より*

lib.rs で全モジュールを pub mod 宣言。error.rs と crypto.rs 以外は空スケルトン。

**実装ファイル**: `core/src/lib.rs`, `core/src/vault.rs`, `core/src/entry.rs`, `core/src/git.rs`, `core/src/legacy.rs`, `core/src/policy.rs`

### 5. cli/src/main.rs 最小構成 🔵

**信頼性**: 🔵 *REQ-004のスケルトン前提*

clap の Parser derive を使い、最小限の main 関数を配置。コマンド定義はTASK-0005で追加。

**実装ファイル**: `cli/src/main.rs`

## 注意事項

- edition, rust-version, license はワークスペース継承を使用

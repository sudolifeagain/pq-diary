## タスク概要

thiserror を使って DiaryError 列挙型を定義する。全Phase (1-4) のエラーバリアント (16種) を含み、Display・Error・From トレイトを実装する。エラーメッセージは英語統一。

## 依存タスク

- **前提タスク**: [TASK-0001: Cargoワークスペース構成](TASK-0001.md)
- **後続タスク**: [TASK-0005: clap CLIスケルトン](TASK-0005.md)

## 完了条件

- [ ] core/src/error.rs に DiaryError 列挙型が定義されている
- [ ] 16種のバリアント: Io, Config, Vault, Entry, Crypto, NotUnlocked, Git, Editor, Search, Import, Template, Legacy, Policy, Daemon, Password, InvalidArgument
- [ ] std::io::Error からの #[from] 変換が機能する
- [ ] 全バリアントの Display が非空の英語メッセージを返す
- [ ] source() でラップ元エラーを返せる
- [ ] 全テストがパスする

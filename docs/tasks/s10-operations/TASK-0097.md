## タスク概要

`docs/definition-of-done.md` に「CLI 整合性」セクションを新規追加する。既存セクション (ビルド・CI / セキュリティ不変条件 / コード品質) に並列で配置し、以下 3 つのチェック項目を含める:

1. 全トップレベルコマンドが `not_implemented` を返さず正常終了することを smoke test で確認
2. 未実装スケルトンは clap 定義から削除、または `#[command(hide = true)]` でヘルプから除外
3. CI に CLI smoke test (`ci/smoke-test.sh` / `.ps1`) が組み込まれている

これにより、今後のスプリント完了時に CLI 表面の整合性 (ヘルプに載っているコマンドはすべて動く) が DoD の一部として保証される。

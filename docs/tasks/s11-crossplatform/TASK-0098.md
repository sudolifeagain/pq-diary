## タスク概要

リポジトリ root に `rust-toolchain.toml` を新規作成し、Rust stable 1.95.0 を固定する。CI と local の rustc バージョン同期を恒久化し、S10 hotfix で表面化した「ローカル 1.94.1 / CI 1.95 stable の clippy 差分」を再発防止する。

## 完了条件

- [ ] `rust-toolchain.toml` がリポジトリ root に存在
- [ ] `[toolchain] channel = "1.95.0"` が記述
- [ ] `[toolchain] components = ["clippy", "rustfmt"]` が記述
- [ ] `[toolchain] profile = "minimal"` が記述
- [ ] `rustup show active-toolchain` がローカルで `1.95.0-...` を返す
- [ ] `cargo build --workspace --locked` が pass (toolchain 切替後の sanity check)

## 実装詳細

`rust-toolchain.toml` (新規):
```toml
[toolchain]
channel = "1.95.0"
components = ["clippy", "rustfmt"]
profile = "minimal"
```

- `channel`: 固定バージョン (ヒアリング Q3)
- `components`: CI で利用する clippy/rustfmt を明示し、初回 rustup 実行時に自動取得
- `profile = "minimal"`: 不要コンポーネントのダウンロードを避けて初回セットアップ時間を短縮

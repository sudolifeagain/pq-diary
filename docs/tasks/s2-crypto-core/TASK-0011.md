## タスク概要

AES-256-GCM暗号化/復号を実装。ランダムnonce(12B)生成、GCMタグ検証。復号結果はSecureBufferで返す。

- [ ] decrypt(key, nonce, ciphertext) → Result<SecureBuffer, DiaryError>

- [ ] 暗号文=暗号化データ+16B GCMタグ

## 実装詳細

### 1. encrypt 関数 🔵

**信頼性**: 🔵 *types.rsより*

OsRng.gen() で nonce 生成 → aes_gcm::Aes256Gcm で暗号化。暗号文は Vec<u8> で返す（暗号文は秘密データではない）。

**実装ファイル**: `core/src/crypto/aead.rs`

### 2. decrypt 関数 🔵

**信頼性**: 🔵 *types.rsより*

aes_gcm で復号 → SecureBuffer で返す（平文は秘密データ）。

**実装ファイル**: `core/src/crypto/aead.rs`

### 3. 設計方針 🔵

**信頼性**: 🔵 *types.rs設計方針より*

- 暗号文は Vec<u8> で返す（暗号文は秘密データではない）
- 復号結果（平文）は SecureBuffer で返す（平文は秘密データ）

## 注意事項

- nonceは毎回OsRngでランダム生成する。nonce再利用はGCMの安全性を破壊する
- GCMタグ(16B)は暗号文に含まれる（aes-gcmクレートのデフォルト動作）

**信頼性**: 🔵 *types.rsより*

### 3. decapsulate 関数 🔵

**信頼性**: 🔵 *types.rsより*

秘密鍵で脱カプセル化。shared_secretはSecureBuffer。

**実装ファイル**: `core/src/crypto/kem.rs`

## 単体テスト要件

### TC-004-01: 鍵ペア生成成功 🔵

**信頼性**: 🔵

**Given**: なし
**When**: keygen() を呼び出し
**Then**: KemKeyPair が正常に返される（encapsulation_key, decapsulation_key が空でない）

### TC-004-02: カプセル化→脱カプセル化で共有秘密一致 🔵

**信頼性**: 🔵

**Given**: keygen() で生成した鍵ペア
**When**: encapsulate → decapsulate
**Then**: 両方の共有秘密が一致する

### TC-004-03: DecapsulationKeyのDrop時zeroize 🔵

**信頼性**: 🔵

**Given**: keygen() で生成したDecapsulationKey
**When**: Drop（スコープ終了）
**Then**: 内部データがゼロ化されている（unsafeテストで確認）

### TC-004-E01: 不正暗号文→脱カプセル化エラー 🔵

**信頼性**: 🔵

**Given**: keygen() で生成した秘密鍵と、不正な暗号文
**When**: decapsulate を呼び出し
**Then**: DiaryError が返る

**テストファイル**: `core/src/crypto/kem.rs` 内の `#[cfg(test)] mod tests`

## 概要

ML-DSA-65によるデジタル署名/検証を実装。秘密鍵はSecureBufferで保護。

## 完了条件

- [ ] DsaKeyPair 構造体 (verifying_key: Vec<u8>, signing_key: SecureBuffer)
- [ ] keygen() → Result<DsaKeyPair, DiaryError>
- [ ] sign(sk, message) → Result<Vec<u8>, DiaryError>
- [ ] verify(pk, message, signature) → Result<bool, DiaryError>
- [ ] 秘密鍵の Drop 時 zeroize
- [ ] 全テストパス

## 実装詳細

1. **keygen**: ml_dsa::MlDsa65 で鍵ペア生成。signing_keyはSecureBuffer 🔵 types.rs + FIPS 204
2. **sign**: 秘密鍵でメッセージに署名 🔵 types.rs
3. **verify**: 公開鍵で署名検証。bool を返す（エラーではなくfalse） 🔵 types.rs

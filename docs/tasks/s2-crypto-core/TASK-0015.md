## 概要

CryptoEngineのunlock()/lock()を実装。Argon2idで鍵導出→検証トークン(AES-GCM)で正当性確認→MasterKeyをSecret<>で保持。lock()でzeroize。

## 完了条件

- [ ] unlock(password, salt, params, verification_iv, verification_ct) → Result<(), DiaryError>
- [ ] lock() で master_key を take() + zeroize
- [ ] 正しいパスワード→unlock成功、is_unlocked()==true
- [ ] 不正パスワード→DiaryError::Crypto
- [ ] 空パスワード→DiaryError::Password
- [ ] lock後に is_unlocked()==false
- [ ] 全テストパス

## 実装詳細

1. **unlock**: derive_key(password, salt, params) → sym_key取得 🔵 dataflow.md
2. **検証トークン復号**: aead::decrypt(sym_key, iv, ct) 🔵 dataflow.md
3. **復号成功→MasterKey構築**: Secret<>で保持 🔵 types.rs
4. **lock**: master_key.take() → Drop時にzeroize 🔵 PRDセクション8.2

> **注意**: Sprint 2ではsym_keyのみ使用。dsa_sk/kem_skの実際の鍵生成はunlock時ではなくinit時に行う（Sprint 3のVault init実装で統合）。S2のunlockではsym_keyの導出と検証トークン確認のみ。

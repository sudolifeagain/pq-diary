## 概要

CryptoEngineに暗号操作メソッド(encrypt/decrypt/kem_*/dsa_*/hmac_*)を統合。施錠状態での操作はNotUnlockedを返す。

## 完了条件

- [ ] encrypt(plaintext) → Result<(Vec<u8>, [u8;12]), DiaryError>
- [ ] decrypt(nonce, ciphertext) → Result<SecureBuffer, DiaryError>
- [ ] kem_keygen() / kem_encapsulate() / kem_decapsulate()
- [ ] dsa_sign() / dsa_verify()
- [ ] hmac() / hmac_verify()
- [ ] 全メソッドが施錠状態でNotUnlockedを返す (ensure_unlocked()チェック)
- [ ] 全テストパス

## 実装詳細

1. **各メソッドの冒頭**: self.ensure_unlocked()? を呼ぶ 🔵 EDGE-103
2. **encrypt/decrypt**: master_key.sym_key を使って aead::encrypt/decrypt に委譲 🔵 types.rs
3. **kem/dsa/hmac**: サブモジュールの関数に委譲 🔵 types.rs
4. **鍵の使用**: dsa_sign は master_key.dsa_sk を使用。kem_decapsulate は master_key.kem_sk を使用 🔵 types.rs

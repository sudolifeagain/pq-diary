## 実装詳細

### 1. read_header 関数 🔵

**信頼性**: 🔵 *REQ-001・types.rsより*

リトルエンディアンでバイナリ読み込み。先頭8バイトのMAGIC検証、バージョンチェックを実施。不正な場合は DiaryError::Vault を返す。

**実装ファイル**: `core/src/vault/reader.rs`

### 2. read_entries 関数 🔵

**信頼性**: 🔵 *types.rsより*

指定されたエントリ数分のEntryRecordをバイナリ読み込み。各レコードのタイプ、サイズ、ペイロードを読み取る。

**実装ファイル**: `core/src/vault/reader.rs`

### 3. read_vault 関数 🔵

**信頼性**: 🔵 *EDGE-001, EDGE-002・types.rsより*

read_header + read_entries を組み合わせたファイル読み込み。パディングは無視して読み飛ばす。

**実装ファイル**: `core/src/vault/reader.rs`
